How to load the Frontend:

1. Clone or download the frontend folder.

2. Make sure you have the following dependencies: npm, node, yarn.

3. Next, enter the dictionary of frontend, and use the command lines:
    $ yarn install
    $ yarn start

4. After 1~2 minute, the fronted code start, and you can see the frontend launch success message:
    'webpack compiled successfully'
 
5. The default browser will automatically open the frontend page.
